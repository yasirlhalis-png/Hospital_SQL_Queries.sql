# SQL Data Management Project

## Overview
This project demonstrates SQL skills through a Hospital Management Database. The project includes 30 SQL queries covering data retrieval, filtering, joins, aggregations, grouping, subqueries, and business insights.

## Skills Demonstrated
- SELECT Statements
- Filtering with WHERE
- ORDER BY
- INNER JOIN
- LEFT JOIN
- GROUP BY
- HAVING
- Aggregate Functions
- Subqueries
- Data Analysis

## Database Tables
- Patients
- Doctors
- Admissions
- Province Names

## Key Insights
- Patient demographics analysis
- Doctor performance tracking
- Admission trends
- Hospital stay calculations
- Province-wise patient distribution
- Diagnosis frequency analysis

## Technologies Used
- SQL
- SQLite
- VS Code
